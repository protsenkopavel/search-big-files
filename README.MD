# File Size Searcher
Приложение для поиска файлов в указанной директории, размер которых превышает заданный порог.
Это полезный инструмент для пользователей, которым нужно быстро найти файлы, которые занимают слишком много места.

## Реализация
Для реализации параллельной обработки файлов в приложении был использован фреймворк ForkJoinPool.
Класс RecursiveFileSearcher, который расширяет RecursiveTask, используется для рекурсивного поиска файлов в директории.
Каждая подзадача обрабатывает свой набор файлов, а затем результаты объединяются вместе.

## Как запустить
Запуск приложения возможен интерактивно с передачей аргументов в консоли, либо в командной строке в формате "String" path, "int" filesize
